# Flashcards de Marinería ⚓

Este es un proyecto web sencillo diseñado para ayudar a estudiantes y entusiastas del mar a memorizar conceptos clave de marinería. Incluye **100 tarjetas interactivas** basadas en cuestionarios reales sobre construcción naval, maniobras, historia y equipos.

## 🚀 Características
* **100 Flashcards:** Cubre desde partes del buque hasta hitos históricos.
* **Interactividad:** Efecto visual de volteo (flip) al hacer clic para ver la respuesta.
* **Diseño Responsivo:** Optimizado para estudiar tanto en ordenadores como en dispositivos móviles.
* **Sin dependencias:** Escrito puramente en HTML, CSS y JavaScript (Vanilla JS). No requiere instalación.

## 🛠️ Cómo usarlo
1. Descarga el archivo `estudio_marineria.html`.
2. Haz doble clic en el archivo para abrirlo en tu navegador web favorito.
3. ¡Empieza a estudiar! Haz clic en la tarjeta para revelar la respuesta y usa los botones para navegar.

## 📚 Contenido de estudio
El temario está dividido en secciones:
1.  Conceptos básicos y dimensiones del buque.
2.  Movimientos y estados (balanceo, cabeceo, etc.).
3.  Maniobras y faenas (filar, fondear, atracar).
4.  Equipos e instrumentos de navegación.
5.  Historia de la navegación.
